This is a simple application designed to validate 'word ladder' puzzles.

A word ladder is a type of word puzzle made up of a series of words, called 'rungs'. You climb up or down the ladder by entering a word, but there is only one correct word for each rung. Your job is to figure out what that word is.

For example feed → feel. The words in the ladder can be almost any length (minimum length of 3 letters) but they must all be the same length. There may be any number of words in the ladder.

Each word varies from the immediately preceeding word by one, and ony one, letter. For example feed → feel. The words in the ladder can be almost any length (minimum length of 3 letters) but they must all be the same length. There may be any number of words in the ladder.

Each rung has a hint that will help you figure out the correct word for that rung.

This page checks for word length, duplicate words, missing rungs (more than one letter changed in a single word pair) and 'bungs'. Bungs are word triplets where the same letter is changed; for example feed → feel → feet. Bungs are allowed, but unnecessary, and are considered bad form.

Enter a word in any of the input fields. As you type, the application will check for word length, missing rungs and bungs. Any warnings will appear to the right of the input field.

Press ESC while editing to undo any edits. ESC only works on the input field currently being edited.

To add a new word to the ladder, enter that word in the last input field and click the 'add' button beside it. A fresh input for the next rung will be added at the end of the list.

To add a word above any existing rungs, click the 'add' button beside that rung. A fresh input will be inserted into the ladder.
